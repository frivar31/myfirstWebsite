# CCML – Centre Culturel Musulman de Longueuil

Site web officiel du Centre Culturel Musulman de Longueuil.

🌐 **Site en ligne** : [mdtarek.online](https://mdtarek.online)

## Structure

```
docs/          ← Source du site (GitHub Pages)
  index.html   ← Page principale
  admin.html   ← Page d'administration (événements)
  events.json  ← Données des événements
  css/         ← Styles
  js/          ← JavaScript (horaires de prières, calendrier)
```

## Fonctionnalités

- Horaires des prières en temps réel (API AlAdhan)
- Calendrier mensuel des prières
- Section événements & annonces
- Page admin pour gérer les événements
- Formulaire de contact
- Don via Zeffy

## Administration

Accédez à `/admin.html` pour gérer les événements.
